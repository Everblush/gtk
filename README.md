## Installation 
```sh 
git clone https://github.com/mangeshrex/Everblush-gtk
npm install -g scss 
sudo make install 
```

## Setup 
- ```gtk-theme-name=Everblush-gtk```
- Add this in your `~/.config/gtk-3.0/settings.ini`
